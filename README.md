Blog CLI Application

This project is a lightweight command-line tool designed to help you manage blog posts directly from your terminal.

Overview
With this application, you can create, view, edit, delete, and search blog posts without relying on a web interface. It’s built to be simple, efficient, and easy to use.

How It Works
The tool runs entirely in the terminal and stores blog posts locally (such as in a file or a small database). Each post contains a title, body content, and optional tags, and can be marked as either a draft or published.

Purpose
This project is ideal for practicing key programming concepts, including:

Building command-line interfaces (CLI)
Processing user input
Implementing CRUD operations (Create, Read, Update, Delete)
Structuring a small, maintainable application
Core Features
Create new blog posts
View a list of all posts
Read a specific post
Edit or delete existing posts
Search posts by keywords or tags

Objective
The aim is to develop a simple yet functional tool that demonstrates fundamental programming skills while remaining practical and easy to expand in the future.
